# AI Lab - InfAU

A hands-on lab where Bauhaus InfAU students use AI-assisted coding to build a web app - no prior web development experience needed.

## What's Inside

| File          | Contents                                                   |
| ------------- | ---------------------------------------------------------- |
| `prompt.md` | Starter prompt - describes the 3D city viewer app to build |
| `CLAUDE.md` | Project guidelines for AI assistants (stack, conventions)  |
| [`docs/cheatsheet.md`](docs/cheatsheet.md) | Quick reference for key concepts (terminal, git, web basics) |
| [`docs/setup.md`](docs/setup.md) | Step-by-step install guide (Git, VS Code, Node.js, Claude Code) |

## Getting Started

1. Follow the [setup guide](docs/setup.md) to install everything
2. Clone this repo and open it in VS Code
3. Start Claude Code and give it the contents of `prompt.md` to build your app
4. Customize and extend it your own way
